# 💻 Tech Price Scout
**A simple tool to find and compare real product prices across India.**

### 🌐 Live Demo
You can try the live version of this app here: [lapzer.streamlit.app](https://lapzer.streamlit.app)

---

### 📖 What is this?
Have you ever searched for a laptop or phone on Google and got confused by "Starting at ₹5,000" only to find out it was just the EMI or a small accessory? 

**Tech Price Scout** is a simple web application that helps you cut through that noise. It searches the web for the product you want, filters out the "fake" low prices (like monthly installments), and shows you a clean list of actual prices from stores like Amazon, Flipkart, and others.

### 🌟 Main Features
* **Live Comparison:** It doesn't use old data. It searches the internet the moment you click the button.
* **Smart Filtering:** You can set a "Minimum" and "Maximum" price. This ensures the results show the actual product and not just its charger or cover.
* **Market Summary:** It automatically calculates the average market price so you know if a deal is actually good or just average.
* **Direct Links:** Every price found comes with a direct link to the store so you can check it out instantly.

### 🛠️ Built With
* **Python:** The engine that runs the logic.
* **Streamlit:** The framework used to create the website interface.
* **Serper API:** The tool that allows the app to fetch Google search results.
